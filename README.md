# 🍦 Ice Cream Parlor Management System

## 📌 About the Project

The Ice Cream Parlor Management System is a desktop application developed using Java and JavaFX. It allows users to customize ice cream orders by selecting the base, size, cone type, and flavors. The system calculates the total price and displays an order summary while demonstrating Object-Oriented Programming principles.

---

## ✨ Features

- User-friendly JavaFX graphical interface
- Cup and Cone selection
- Multiple ice cream sizes
- Different cone types
- Multiple flavor selection
- Automatic price calculation
- Order summary generation
- File handling for storing orders

---
